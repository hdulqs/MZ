注册登录API

用于用户的登录注册

接口 描述
http://api.hurong.com/exapi/api/index/applogin 用户登录(身份验证)
http://api.hurong.com/api/user/register 用户注册

配置API

获取hurong最新市场配置数据

接口 描述
http://api.hurong.com/data/v1/markets 获取已开启的市场信息，包括价格、数量小数点位数

行情API

获取hurong最新市场行情数据

接口 描述
http://api.hurong.com/data/v1/ticker 行情
http://api.hurong.com/data/v1/depth 市场深度
http://api.hurong.com/data/v1/trades 历史成交
http://api.hurong.com/data/v1/kline K线

交易API

用于hurong快速进行交易

接口 描述
https://trade.hurong.com/api/order 委托下单
https://trade.hurong.com/api/cancelOrder 取消委托
https://trade.hurong.com/api/getOrder 获取委托买单或卖单
https://trade.hurong.com/api/getAccountInfo 获取用户信息
https://trade.hurong.com/api/getUserAddress 获取用户充值地址
https://trade.hurong.com/api/getWithdrawAddress 获取用户认证的提现地址
https://trade.hurong.com/api/getWithdrawRecord 获取数字资产提现记录
https://trade.hurong.com/api/getChargeRecord 获取数字资产充值记录
https://trade.hurong.com/api/withdraw 提现

错误代码

所有API方法调用在请求失败或遇到未知错误时会返回JSON错误对象。

代码 描述
1000 调用成功
1001 一般错误提示
1002 内部错误
1003 验证不通过
1004 资金安全密码锁定
1005 资金安全密码错误，请确认后重新输入。
1006 实名认证等待审核或审核不通过
1009 此接口维护中
2001 人民币账户余额不足
2002 比特币账户余额不足
2003 莱特币账户余额不足
2005 以太币账户余额不足
2006 ETC币账户余额不足
2007 BTS币账户余额不足
2009 账户余额不足
3001 挂单没有找到
3002 无效的金额
3003 无效的数量
3004 用户不存在
3005 无效的参数
3006 无效的IP或与绑定的IP不一致
3007 请求时间已失效
3008 交易记录没有找到
4001 API接口被锁定或未启用
4002 请求过于频繁
